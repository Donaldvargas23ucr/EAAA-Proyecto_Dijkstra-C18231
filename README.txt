Para el correcto funcionamiento de este codigo se necesitan los archivos "sfml-graphics-2.dll", "sfml-system-2.dll", "sfml-window-2.dll" y "openal32.dll".
Dado que son los que generan la imagen del grafo, así como también la version SFML 2.5.0 y la fuente arial.ttf, si se desea incluir un grafo en texto
debe ser ingresado en un archivo llamado "grafo1.txt".
